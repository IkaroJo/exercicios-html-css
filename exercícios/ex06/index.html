Principais Formatações

Negrito/Destaque

Nessa frase, temos o texto em negrito/destaque

Para fazer isso, se utiliza a tag <b>, mas é uma tag não semântica, sem significado

Para se fazer uma frase em negrito/destaque, semanticamente correta, se utiliza a tag <strong>


Itálico/Ênfase

Nessa frase, temos o o texto em Itálico/ênfase, utilizando a tag <i>. Essa teg não é semântica, não tem significado, somente uma forma

Nesse texteo temos a frase em itálico/ênfase utilizando a tag <em>, que possui uma semântica

As tag <b> e a tag <i> são tag se formas. Já as tag <strong< e a tag <em>, são tag de significado, strong significa forte ou urgência e EM significa ênfase
