Principais Formatações

Negrito/Destaque

Nessa frase, temos o texto em negrito/destaque

Para fazer isso, se utiliza a tag <b>, mas é uma tag não semântica, sem significado

Para se fazer uma frase em negrito/destaque, semanticamente correta, se utiliza a tag <strong>


Itálico/Ênfase

Nessa frase, temos o o texto em Itálico/ênfase, utilizando a tag <i>. Essa teg não é semântica, não tem significado, somente uma forma

Nesse texteo temos a frase em itálico/ênfase utilizando a tag <em>, que possui uma semântica

As tag <b> e a tag <i> são tag se formas. Já as tag <strong< e a tag <em>, são tag de significado, strong significa forte ou urgência e EM significa ênfase

Texto Marcado

Podemos marcar um texto, desse jeito, utilizando a tag <mark>

Texto grande e pequeno

Nesse parágrafo, iremos estudar sobre textos grandes e textos pequeno Iremos utilizar a tag <big> e a tag <small>. A tag <big> é obsoleta, pois era uma tag de forma e não semântica.

Texto deletado

Podemos utilizar essa marca para um texto como excluído, para indicar que ele foi lido mas não considerado. Para isso, se utiliza a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para isso, se utiliza a tag <ins>

Texto sobrescrito

Para indicar coisas do tipo x20+3. Utilizando a tag <sup>

Texto subscrito

Para inserir coisas tipo H2O. Utilizando a tag <sub>
