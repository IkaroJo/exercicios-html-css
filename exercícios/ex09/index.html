Usando Links Externos

Podemos utilizar links externos utilizando a tag <a>

Dessa forma, podemos associar o texto em a um link externo dessa forma: esse é o meu GitHub

Utilizando essa forma, ao clicar no link, a página será redirecionada para a URL que você escolheu. Para que abra uma nova aba, é necessário utilizar as marcas target e a marca rel dentro da tag <a>: esse é o meu GitHub
